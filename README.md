# gulp-config v2

## Что умеет данная конфигурация

---

#### В режиме разработчика
###### ___Для запуска необходимо ввести команду: `gulp` или `npm run dev`___

Локальный сервер `browser-sync` автоматически обновляет страницу при изменении данных в файлах.

Копирование файлов из папки `src` в папку `dist`.

Удаление и повторное создание папки с результатом(`dist`) при запуске gulp. Очистка папки от файлов которые были ранее удалены из исходной папки(`src`).


Наблюдение за всеми файлами и папками и автоматическое обновление всей структуры проекта.

Создает ключ в файле `version.json`, который запрещает кешировать сайт в браузере. Позволяет видеть все изменения в проекте.

Собирает файлы в формате `js` таким образом , что бы можно было экспортировать и импортировать в синтаксисе `ES6`

Автоматически конвертирует шрифты из иcходной папки(`src`) в папку с результатом(`dist`), а так же записывает их подключение в `scss` файле

---

#### В режиме продакшена
###### ___Для запуска необходимо ввести команду:  `npm run build`___

Копирование файлов из папки `src` в папку `dist`.


Сборка HTML-файлов из исходной папки(`src`) в папку с результатом(`dist`) в один файл index.html.

Решение проблемы с путями при подключении картинок во вложенных  HTML-файлах. Алиас(короткое имя) - `@img`, заменяет `${folder}/src/img`. Плагин `gulp-file-include` в файле index.html `@img`, заменяет на `img/`.

Подключает альтернативные картинки в формате `webp` по дефолту, если браузер не поддерживает такой формат - то `jpeg`.


Помогает обработать возникающие ошибки с тем или иным файлом в терминале и выводит сообщения об ошибках в операционных системах.



Компилирует файлы стилей формата `scss` из исходной папки(`src`) в привычный браузеру `css` в папку с результатом(`dist`).

Минифицирует файлы стилей и скриптов в папке с резутьтатом, создает файлы `min.css`, `min.js`.

Автоматически добавляет вендорные префиксы в файлах стилей для поддержки в старых браузерах.

Группирует медиа запросы в файлах стилей.

Конвертирует изображения в формат `webp` из исходной папки(`src`) в папку с результатом(`dist`). Также сжимает и оптимизирует картинки без потери качества.
